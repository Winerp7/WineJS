For at køre dev server imens du udvikler
så skriv i terminalen:
  npm run dev
det fyrer op for en dev server
som genstarter, hver gang der laves en ændring
så du skal bare refresh "localhost:3000" for at se din ændring på siden.

Struktur:
app.js er applikationen som bliver kørt
Views --> alt html og UI
Routes --> alle URLS som skal håndteres, de kalder funktioner som udføre backend logikken


Ressources:
https://html2jade.org - converst html to pug/jade
https://html2pug.now.sh - converst html to pug/jade
